# 课堂手机管理系统 - 实验报告

## 一、需求分析

### 1.1 项目背景
学生在课堂上使用手机影响教学效果，需要开发课堂手机管理系统规范课堂秩序。

### 1.2 功能需求
- 教师角色：查看课堂、登记没收手机、管理记录、查看统计报表
- 统计报表：按班级、教师、课程、周次、节次、时间、状态等维度统计
- 基础数据：学期、班级、教室、教师、课表信息管理

## 二、数据库设计

### 2.1 主要表结构

| 表名 | 说明 |
|------|------|
| sys_schedule | 课表（教师、班级、教室、课程、周次、节次、日期） |
| phone_confiscate_record | 没收记录（课表ID、班级ID、教师ID、数量、时间、状态） |

### 2.2 关键字段
- `phone_confiscate_record`: id, schedule_id, teaching_class_id, teacher_id, confiscate_count, confiscate_time, handover_status

## 三、业务流程

1. 教师查看近两日授课课堂
2. 选择课堂登记没收手机
3. 查看、修改、删除没收记录
4. 查看多维度统计报表

## 四、数据库操作（增删改查）

### 4.1 增加
```java
recordService.save(record);
```

### 4.2 查询
```java
recordService.list(wrapper);
```

### 4.3 修改
```java
recordService.updateById(record);
```

### 4.4 删除
```java
recordService.removeById(id);
```

## 五、技术实现

- Spring Boot 3.2.10 + Java 17
- MyBatis-Plus 3.5.8
- Thymeleaf 模板引擎
- MySQL 数据库
